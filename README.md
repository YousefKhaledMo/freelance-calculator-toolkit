# Freelance Calculator Toolkit

A comprehensive freelance pricing intelligence toolkit for freelancers, contractors, and consultants.

## Features

- **Price Calculator** - Calculate optimal project pricing based on hours, complexity, experience, and project type
- **Scope Estimator** - Estimate project scope and timeline requirements
- **Profit Calculator** - Calculate profit margins and net income
- **Workload Planner** - Plan and estimate workload distribution
- **Discount Calculator** - Calculate discounted pricing with visual sliders

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No external dependencies
- Responsive design with dark/light themes
- Google Fonts (Space Grotesk, Space Mono)

## Usage

Simply open `index.html` in any modern web browser to use the toolkit.

## Supported Currencies

USD, EUR, GBP, EGP, INR, AED, SAR, CAD, AUD, CHF, JPY, BRL, MXN, KWD
